
# Personal Dev Setup !!

This dotfiles can be install on your system Linux.  
It contain:


- VIM + confs
- NVIM + confs
- Tmux + confs

---

and many base tools:

- build-essential 
- curl wget 
- git 
- tree 
- htop 
- unzip zip 
- ca-certificates 
- software-properties-common 
- tmux 
- vim-gtk3 
- xclip 
- python3 
- python3-pip 
- python3-venv


and debug / network tools:

- strace 
- ltrace 
- gdb 
- lsof 
- net-tools 
- iproute2 
- netcat-openbsd 
- tcpdump

--- 

## How to install 


```bash 
git clone https://github.com/Dr00pyd00/dotfiles ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

> ⚠️ If config files already exist on your system, they will be automatically backed up with a `.bak` extension before being replaced.
